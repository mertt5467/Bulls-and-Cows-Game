# Bulls and Cows Game (ENG)

A terminal-based Mastermind (Bulls and Cows) game written in Java. Challenge your logic by cracking the hidden numerical code.

## Installation and Usage

This is a Java source code. You need to have Java JDK installed on your computer to compile and run it.

### 🐧 Linux / 🪟 Windows (Command Line)

1. **Clone the project:**
   ```bash
   git clone https://github.com/mertt5467/Bulls-and-Cows-Game.git
   cd Bulls-and-Cows-Game
   ```

2.  **Compile the code:**
    ```bash
    javac App.java
    ```

3.  **Run the application:**
    ```bash
    java App
    ```

# Bulls and Cows Oyunu (TR)

Bu bir Java kaynak kodudur. Çalıştırmadan önce bilgisayarınızda Java JDK kurulu olması gerekir.

### 🎮 Gameplay / Oynanış

**Bull (Boğa):**

Correct digit in the correct position. / Doğru rakam, doğru yerde.

**Cow (İnek):**

Correct digit but in the wrong position. / Rakam doğru ama yeri yanlış.
